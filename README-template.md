### What I learned

During this project I learned how to combine CSS Grid and Flexbox to build complex layouts. I also improved my understanding of responsive design using media queries and converting px values into rem units.


```css
@media (max-width:48rem){
   .bento-grid{
      display:flex;
      flex-direction:column;
   }
}
```

### Continued development

I want to improve my responsive design skills and become more comfortable creating layouts that adapt smoothly across different screen sizes.
